## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/tdnhome/pyspark-examples.git
```

---

### 2. Set Up Environment

We recommend using jupyter running on local docker:

* Start Jupyter
    - Start Docker Desktop
    - $ docker run -p 8888:8888 -v /jupyter/notebooks:/home/jovyan/work jupyter/pyspark-notebook
* Note -v: volume mapping from local host to container
    - Host folder: /jupyter/notebooks
    - Container folder: /home/jovyan/work
* Browser: http://localhost:8888
    - Check the Jupyter log for token
